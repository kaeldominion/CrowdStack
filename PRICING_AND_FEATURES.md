### Account & Profile
- **Email/Passwordless Authentication**: Magic link login system
- **Progressive Profile Creation**: Multi-step Typeform-style signup
  - First registration: Email, First Name, Last Name, Gender, Instagram
  - Subsequent registrations: Additional profile fields
- **Profile Management Dashboard**: View and edit personal information
- **Avatar Upload**: Profile picture management with image optimization
- **Social Links**: Instagram, TikTok handles
- **Bio & Personal Information**: Date of birth, WhatsApp number
- **Event History**: View past and upcoming events
- **XP & Gamification System**: Earn XP for various activities
  - Event attendance (100 XP per check-in)
  - Early registration (25 XP bonus within 24h)
  - Repeat venue attendance bonuses
  - Profile completion rewards
  - Social connection bonuses
  - Referral sharing bonuses

### Event Discovery & Registration
- **Public Event Browsing**: Browse all published events
- **Event Landing Pages**: Rich event pages with cover images, descriptions, venue details
- **Event Registration**: Multi-step registration form (no password required)
- **QR Pass Generation**: Unique QR codes for event entry
- **Registration Management**: View, download, and share QR passes
- **Deregistration**: Remove yourself from events
- **Event Sharing**: Share events with friends via referral links
- **Add to Calendar**: Calendar integration for events

### Social & Engagement
- **Referral Tracking**: Track friends you've referred to events
- **Share & Earn XP**: Share events with friends and earn XP rewards
- **Photo Gallery Access**: View event photo albums after events
- **DJ Profiles & Follows**: Follow DJs, view profiles and mixes (if DJ feature enabled)
- **Event Lineups**: View DJ/artist lineups for events

### Mobile Experience
- **Responsive Design**: Mobile-first design for all features
- **QR Code Display**: Mobile-optimized QR pass viewing
- **Push Notifications**: (Future) Event reminders, photo notifications
